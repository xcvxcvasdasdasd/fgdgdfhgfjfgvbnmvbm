## Описание

Training Planner — это GUI‑приложение для планирования и учёта тренировок. Программа позволяет:
* добавлять тренировки с указанием даты, типа и длительности;
* фильтровать записи по типу тренировки и дате;
* удалять выбранные тренировки;
* сохранять данные в формате JSON;
* загружать ранее сохранённые данные при запуске;
* проверять корректность вводимых данных.

## Функциональные возможности

* **Добавление тренировок** с валидацией данных.
* **Фильтрация** по типу тренировки и конкретной дате.
* **Сброс фильтров** для отображения всех записей.
* **Удаление выбранных тренировок** из списка.
* **Автоматическое сохранение** данных в `trainings.json`.
* **Загрузка данных** при запуске приложения.
* **Статус‑бар** с отображением количества тренировок.
* **Гибкая валидация даты** — поддержка трёх форматов ввода.

## Использование
### Добавление тренировки
1. Введите дату в одном из форматов:

- ГГГГ-ММ-ДД (например, 2024-05-15);

- ДД.ММ.ГГГГ (например, 15.05.2024);

- ДД/ММ/ГГГГ (например, 15/05/2024).

2. Выберите тип тренировки из выпадающего списка или введите свой вариант.

3. Укажите длительность тренировки в минутах (положительное целое число).

4. Нажмите кнопку «Добавить тренировку».

### Фильтрация данных
1. Выберите тип тренировки в фильтре «Фильтр по типу» (или оставьте «Все»).

2. Введите дату в формате ГГГГ-ММ-ДД в поле «Фильтр по дате» (опционально).

3. Нажмите «Применить фильтры» для отображения отфильтрованных данных.

4. Нажмите «Сбросить фильтры», чтобы показать все записи.

### Удаление тренировки
1. Выделите нужную запись в таблице (клик левой кнопкой мыши).

2. Нажмите кнопку «Удалить выбранную».

3. Подтвердите удаление — появится сообщение об успехе, запись исчезнет из таблицы и файла данных.

### Валидация данных
Приложение проверяет:

- Формат даты — поддерживает три формата: ГГГГ-ММ-ДД, ДД.ММ.ГГГГ, ДД/ММ/ГГГГ. При ошибке выводится сообщение.

- Длительность — должна быть положительным целым числом. При вводе отрицательного числа или текста выводится сообщение об ошибке.

- Тип тренировки — может быть любым текстом. Рекомендуется выбирать из предложенных вариантов: «Кардио», «Силовая», «Йога», «Растяжка».
